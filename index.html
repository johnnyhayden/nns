Format:
  • Section labels: V:, C:, B:, I:, TA:, Solo: - colon required (up to 4 chars)
  • Numbered sections: V1:, C2:, V2:, etc. - renders in boxes
  • Type any number of chords - auto-wraps into 4-beat bars (e.g., 12 chords = 3 lines)
  • Auto-spaces when typing numbers (type "1451" â†’ get "1 4 5 1")
  • Use / for rhythm hits: 1 / / /
  • Held chords: <2> displays as diamond around number
  • Tied chords: 1_4_5_1 displays underlined
  • Comments: #Your comment here (appears below chords)
  • Repeat markers: ||:, :||
  • Chords: 1, 2-, 3, 4, 5, 6-, 7o (- for minor, o for diminished)
  • Seventh chords: 17, 4-7, 57 (adds superscript 7)
  • Suspended: 1s, 4sus, 5sus2, 2sus4 (adds superscript)
  • Inversions: 5/1, 4/2 (renders as stacked fraction)
  • Push early: <4 (shows < above chord)
  • Push late: 5> (shows > above chord)
  • Push diamonds: <<5> (early), <5>> (late)
  • Staccato accent: 5^ (shows ^ above chord)
  • Beat ticks: 1'_4''' (1 beat on 1, 3 beats on 4 - ticks shown above)
